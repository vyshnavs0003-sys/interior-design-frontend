# Interior Design Company Website

Academic Frontend Development Project  
Calicut | December 2025

## Project Overview
A fully responsive, multi-page frontend website developed for an interior design company.  
The project focuses on modern UI design, clean layouts, and intuitive navigation.  
This project was created strictly for educational purposes only.

## Technologies & Tools Used
- Markup & Styling: HTML5, CSS3
- Frontend Development: JavaScript (ES6), Bootstrap
- Data Handling: Fetch API, Local JSON

## Website Pages
The website includes the following pages:
- Home
- About
- What We Do
- Products
- Offers
- Gallery
- FAQ
- Contact

## Key Features
- Fully responsive design across desktop, tablet, and mobile devices
- Consistent and modern user interface design
- User-friendly navigation across multiple pages
- Optimized assets and organized code structure
- Maintainable and well-documented frontend codebase

## Development Details
- Designed and implemented layouts for all pages using semantic HTML and CSS
- Applied responsive design techniques using media queries and flexible grid layouts
- Used CSS for typography, color schemes, spacing, and visual hierarchy
- Integrated JavaScript for interactive elements such as navigation behavior and form validation

## Dynamic Content Implementation
- Used the Fetch API to retrieve data from local JSON files
- Dynamically generated card-based UI components using JavaScript
- Implemented asynchronous data loading with proper error handling
- Ensured responsive rendering of dynamically loaded content
